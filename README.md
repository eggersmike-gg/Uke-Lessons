# Uke Lessons

A React app for tracking ukulele lessons across multiple online subscriptions.

## Features

- Track lessons with title, URL, description, source, status, rating, and notes
- Filter by status or source, search across all text fields, sort by date/title/rating
- Stats bar showing total / completed / in-progress / not started counts
- Manage your subscription sources (Fender Play, Yousician, etc.)
- Export and import all data as JSON
- Data persisted in `localStorage` — no backend required

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```
